Аннотация проекта

Проект «Сервисы для профсоюзной организации» — это проект по дисциплине «Проектная деятельность», нацеленный на создание для профсоюзной организации Московского Политеха обновленного веб-сайта, который может быть полезен как студентам университета, так и его работникам.

Сайт служит сборником полезной информации и нормативно-правовых актов внутри высшего учебного заведения и позволяет его обучающимся и работникам в полной мере раскрыть потенциал профсоюзной организации.

О проекте

Структура сайта

  • Главная - стартовая страница с основной информацией
  • Документы - нормативно-правовые акты
  • Справочник - полезная информация
  • Проекты - реализованные инициативы
  • Команда - состав профсоюзной организации
  • Новости - актуальные обновления

Особенности

  • Кнопка "Вступить в профсоюз" в шапке сайта
  • Дизайн, адаптированный под десктопную и мобильную версии
  • Анимации при наведении
